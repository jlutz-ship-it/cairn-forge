FORGE COMMS HUB — Operator Manual

    

      This is Jordan's relay command center. Every message between builders flows through here.
      Builders are separate conversations — they cannot talk directly. You are the router.
    


    
Keyboard Shortcuts

    

      
Ctrl+EnterSend message

      
Ctrl+Shift+EnterBroadcast to all builders

      
Ctrl+HToggle this help

      
EscapeClose modals

      
1-9, 0Select builder (when not typing)

    


    
Workflow

    

      
1. RECEIVEBuilder gives you output → paste it in the "Parse Incoming" box → click PARSE

      
2. ROUTEHub auto-detects sender, type, and recipient from the header

      
3. COMPOSEWrite new messages with the compose bar — auto-formats with Forge Comms Standard

      
4. COPYClick "Copy Formatted" on any message → paste into target builder's conversation

      
5. BROADCASTClick BROADCAST TO ALL → generates formatted text → copy once, paste to everyone

    


    
Message Types

    

      
BROADCASTInformation for everyone. No action required.

      
DIRECTIVEAn order. Action required from recipient.

      
REPORTStatus update. Build complete, progress, etc.

      
BUILD LOGDetailed record of what was built.

      
QUESTIONNeeds an answer from a specific builder or Jordan.

      
APPENDIXReference material. Read and file.

    


    
The Three Rules

    

      
1. SIGNEvery message has FROM, ROLE, DATE, TO, TYPE header + signature

      
2. BROADCASTWhen Jordan teaches one builder, that teaching goes to ALL during the shift

      
3. RESEARCHInternet research before building. Proven pattern.
